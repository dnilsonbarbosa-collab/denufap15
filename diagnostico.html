📖 Como Instalar Corretamente

  1. Use o Chrome - Edge/Firefox criam apenas atalhos
  2. Acesse via HTTPS
  3. Toque nos 3 pontos → "Instalar aplicativo"
  4. Verifique a gaveta de apps - App deve aparecer lá
  5. Ícone limpo - Sem badge do Chrome no canto
